Diabetes data cleaning, thana-level distribution, age-group patterns, and control status mapping

Author

Miskatul Anwar

Published

2026-06-05

Overview

This project analyzes diabetes patient data from the Chittagong region using a reproducible Python and Quarto workflow. The work covers data cleaning, fasting glucose standardization, thana-level distribution analysis, City Corporation classification, age-group summaries, and spatial mapping.

The project focuses on preparing clean, analysis-ready clinical data and producing regional summaries for Chattogram City Corporation and outside-City-Corporation areas.

Project Goals

  • Clean and standardize raw diabetes patient records.
  • Update fasting glucose values using supplementary patient-level data.
  • Standardize thana names for reliable regional analysis.
  • Compare diabetes patient distribution inside and outside Chattogram City Corporation.
  • Analyze age-group patterns using <=30 and >30 categories.
  • Map thana-level diabetes distribution and control status.
  • Generate reproducible outputs for reporting and future analysis.
